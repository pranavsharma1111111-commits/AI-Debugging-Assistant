# AI Debugging Assistant

An AI-powered multi-language debugging assistant built using Flask and Gemini API.

This project helps developers detect coding errors and generate corrected code using AI.

---

# Features

* Multi-language debugging support
* AI-generated error explanations
* Corrected code suggestions
* Clean dark-themed UI
* Loading state while analyzing
* Secure API key handling using `.env`
* Beginner-friendly explanations

---

* Python

* C#

# Installation

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/AI-Debugging-Assistant.git
```

---

## 2. Install Dependencies

```bash
pip install flask flask-cors google-genai python-dotenv
```

---

## 3. Create `.env` File

Create a `.env` file inside the project folder and add:

```env
GEMINI_API_KEY=your_api_key_here
```

---

## 4. Run Application

```bash
python app.py
```
